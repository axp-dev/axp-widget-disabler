== Description ==

The plugin allows you to disable certain widgets

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'AXP Widget Disabler'
3. Activate AXP Widget Disabler from your Plugins page. (You will be greeted with a Welcome page.)

= From WordPress.org =

1. Download AXP Widget Disabler.
2. Upload the 'AXP Widget Disabler' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate AXP Widget Disabler from your Plugins page. (You will be greeted with a Welcome page.)
